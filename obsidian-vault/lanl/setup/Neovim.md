# Neovim
```bash
  git clone git@github.com:Neovim/neovim.git
  cd neovim
  make CMAKE_BUILD_TYPE=Release CMAKE_INSTALL_PREFIX=/opt/nvim
  sudo make install /opt/nvim
```

# Tree-Sitter CLI
In ~/.config/nvim/local
```bash
  npm install tree-sitter-cli
```
