# MacOS Setup

## Spacers in dock

Normal spacer:
```shell
$ defaults write com.apple.dock persistent-apps -array-add '{"tile-type"="spacer-tile";}'; killall Dock

```

Space to right:
```shell
$ defaults write com.apple.dock persistent-others -array-add '{"tile-type"="spacer-tile";}'; killall Dock
```
